# AI for Personal Productivity

A mini course on using AI to manage your daily life more effectively.

## Course Overview

This course teaches practical ways to use AI for everyday tasks. Learn how to handle email faster, stay organized, learn new skills, and build habits that stick.

## Lessons

1. **AI for Daily Life** - Introduction to using AI for personal productivity
2. **Email and Communication** - Handle email and messages more efficiently
3. **Organization and Planning** - Use AI for projects, to-do lists, and decisions
4. **Learning New Skills** - AI as your patient, always-available tutor
5. **Building Habits** - Create sustainable AI habits that stick

## Structure

- `video-scripts/` - Scripts for each lesson video
- `docs/` - Lesson materials with AI Chat Exercises

## AI Chat Exercises

Each lesson includes an interactive AI Chat Exercise with a system prompt designed to give learners hands-on experience with the concepts covered.
