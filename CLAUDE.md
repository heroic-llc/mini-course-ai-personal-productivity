# AI for Personal Productivity - Course Guide

## Course Context
This is a mini course for writingIO focused on using AI for personal productivity and daily life management.

## Video Script Guidelines
- Target length: 1-2 minutes per video (roughly 900 characters per minute)
- Use "A I" (with space) for proper text-to-speech pronunciation
- Keep language conversational and practical
- Focus on actionable tips, not theory

## AI Chat Exercise Format
Each lesson doc contains:
- Overview of the lesson topic
- System prompt for the AI Chat Exercise
- Learning objectives

The system prompts are designed to be used directly with AI models to create interactive learning experiences.

## Lesson Topics
1. AI for Daily Life - Introduction and overview
2. Email and Communication - Drafting, summarizing, handling difficult messages
3. Organization and Planning - Projects, to-do lists, decisions, meal and travel planning
4. Learning New Skills - AI as a patient tutor
5. Building Habits - Starting small, creating triggers, building templates
